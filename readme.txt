Python version:
   Python 3.x (3.4+)

Requirements are specified in requirements.txt. To install them:
   pip3.x install -r requirements.txt

   (where 3.x is the Python version you're using.)

   

Install NLTK punkt data:
   
   $ python
   >>> import nltk
   >>> nltk.download()

Choose download, and specify "punkt" data.
